# Netflix Data Science Project 🎬📊

A complete end-to-end data science project analyzing Netflix data — from data collection and cleaning to exploratory data analysis, visualization, and machine learning. This project was built while following a YouTube tutorial and adapted/extended for personal learning.

## 📌 Project Overview

This project dives into the Netflix dataset to uncover trends, patterns, and insights about content available on the platform. It covers the full data science workflow:

- ✅ **Data Collection & Cleaning** — Preparing raw Netflix data for analysis
- ✅ **Exploratory Data Analysis (EDA)** — Identifying trends, patterns, and insights
- ✅ **Data Visualization** — Creating charts and graphs to present findings
- ✅ **Machine Learning** — Predictive modeling for personalized recommendations
- ✅ **Insights & Conclusions** — Key takeaways and actionable findings

## 🗂️ Dataset

The dataset used in this project contains information about Netflix shows and movies (title, cast, director, country, release year, rating, duration, genre, etc.).


## 🛠️ Tech Stack

- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
- **Tools:** Jupyter Notebook

## 🚀 Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/netflix-data-science-project.git
   cd netflix-data-science-project
   ```
2. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```
3. Run the notebooks in order (data cleaning → EDA → visualization → modeling)
